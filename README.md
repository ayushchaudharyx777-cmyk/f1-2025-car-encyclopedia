## ✨ Features

| Feature | Description |
|---|---|
| 🎮 **3D animated cars** | Perspective-based hover-tilt with continuous idle float animation per team |
| 📡 **Live telemetry simulation** | Speed, RPM, gear, ERS, brake temp, tyre wear updating in real time |
| 🔧 **Pit stop simulation** | Full multi-stage sequence with live countdown timer |
| 🌧️ **Weather modes** | Toggleable rain effect per car |
| 🔥 **Brake glow effect** | Dynamic glow on braking state |
| ⚡ **DRS toggle** | Per-car DRS activation state |
| 📱 **Fully responsive** | Sticky nav, scroll-reveal animations, mobile-friendly grid layouts |

## 🚀 Run Locally

```bash
git clone https://github.com/ayushchaudharyx777-cmyk/f1-2025-car-encyclopedia.git
cd f1-2025-car-encyclopedia
```

Fir seedha `index.html` ko browser me open kar do — bas double click karo. Agar live-reload chahiye (code change karte waqt auto refresh), to VS Code me **Live Server** extension install karke "Go Live" pe click karo.

> 💡 Tip: Sirf double-click se bhi chal jaata hai kyunki ye pure HTML/CSS/JS hai — koi build step ya `npm install` nahi chahiye.

## 🔮 Roadmap / Future Aspects

Ye project abhi bhi evolve ho raha hai — aage kya aane wala hai:

- 🏁 **Live F1 API integration** — real-time lap times, positions aur sector splits
- 🖼️ **True 3D car models** — Three.js ke saath CSS perspective tricks ko replace karna
- 🎧 **Engine sound effects** — throttle/telemetry state ke saath sync karke
- 📊 **Historical season comparison** — 2023 vs 2024 vs 2025 car specs side-by-side
- 🌓 **Light/Dark theme toggle**
- 🌍 **Circuit map view** — har Grand Prix ka track layout dikhana
- 📱 **PWA support** — mobile pe offline access ke liye
- 🏆 **Driver standings & championship tracker**
- 🎙️ **Team radio snippets / fun facts popup** per car
- 🧠 **AI-based strategy predictor** — tyre wear + weather ke basis pe pit strategy suggest karna
